Left Side Navigation Demo

Hover over the left edge of the screen to reveal the navigation menu.

The menu automatically hides when you move your cursor away.

Features:

  • Auto-hide on mouse leave
  • Smooth slide-in/out animation
  • Responsive design
  • Modern glassmorphism effect
  • Icon-based navigation with labels

How to Use:

  1. Copy the CSS from left-side-nav.css
  2. Copy the HTML navigation structure
  3. Paste both into your existing pages
  4. Update the href links to match your pages
